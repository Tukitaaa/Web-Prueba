

# Tuka Shop (demo)

Demo moderno de una tienda de cursos llamada Tuka Shop. Incluye:

- Listado de cursos
- Búsqueda y filtros por categoría
- Carrito con persistencia en `localStorage`
- Modal de detalles con sistema de reseñas (persistente en `localStorage`)
- Animaciones de entrada al desplazarse

Cómo usar

1. Abrir `index.html` en el navegador o servir con un servidor estático.
2. Buscar cursos, filtrar y añadir al carrito.
3. En "Ver detalles" puedes ver y agregar reseñas por curso.
4. El pago sigue siendo simulado.

Archivos principales:

- `index.html` – página principal y modales.
- `styles.css` – estilos modernos y animaciones.
- `app.js` – lógica, carrito y reseñas.
